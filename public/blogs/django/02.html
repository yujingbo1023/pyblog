    • 在django1版本用的多，因为之前没有动态路由的写法。

                   path('edit/', views.login),
                   path('list/', views.login),
               ], None, None)),

def login(request):

    # 4.如果post请求，传递请求体（原始数据），使用postman发送raw类型的json数据
    # 默认发送不了，因为了django中默认有csrf机制，在setting中注释掉（中间件部分）
